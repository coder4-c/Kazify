# EYOB (local scaffold)

Minimal scaffold added: an Express backend and a Vite + React frontend.

Quick start (PowerShell):

1. Backend

```powershell
cd "c:\Users\user\Downloads\EYOB\Backend"
npm install
npm run start
```

2. Frontend

```powershell
cd "c:\Users\user\Downloads\EYOB\Frontend"
npm install
npm run dev
```

Endpoints

- Backend health: http://localhost:4000/api/health
- Backend hello: http://localhost:4000/api/hello
- Frontend dev (Vite): typically http://localhost:5173
